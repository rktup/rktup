rktup

Easy rkt ACI discovery

rkt's App Container Image (ACI) Discovery locates image and signature URLs using HTTPS+HTML. rktup.org can be used to easily configure image and signature URLs from repositories on GitHub.

Usage

Place a file .rktup.json with the following structure in the root of your GitHub repository or any subfolder. pubkey is optional.

Example


    {
      "discovery": {
        "url_template": "https://sos.exo.io/my-acis/my-project-{version}-{os}-{arch}.{ext}"
      },
      "pubkey": {
        "url": "https://sos.exo.io/my-acis/signing-key.asc
      }
    }

Use rkt to fetch or run your image, for example:


    rkt run --interactive rktup.org/my-user/my-project

Caveats

rkt verifies the app name on download, i.e. the name must be set to the rktup name or --insecure-options=image must be used. rkt image list does show the image by its name as set in the image manifest.
